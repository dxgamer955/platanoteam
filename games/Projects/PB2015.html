How To Play:

Use the following keys to control your character:

  • Arrow Keys - Move
  • X - A Button
  • Z - B Button
  • S - X Button
  • A - Y Button
  • Enter - Start
  • Shift - Select
  • Q - L Button
  • W - R Button

Objective: Complete each level by tapping all Jovani Vazquez!
